Week 2 Notes

Programming Basics

Declaring and Assigning

All variables have to be declared before they can be used, the '=' is the operator used to do this Variables can be defined by either using 'let' or 'const', let means the variable can be changed (for example a high score), while const means it cannot be changed or reassigned (for example using pi in a calculation).

The keyword const is used when the variable will not be reassigned to another value, whereas let is used if the variable might be reassigned later in the program.


You can even declare and assign multiple variables at the same time if you separate them with commas:
let x = 3, y = 4, z = 5;


Constructor Function

Let's say you are building a game like Sims, everything in the game with characteritics could be created as a cnstructor function. So, for example a person, think of all the things you could customize for each person: name, age, hair color, eye color, hairstyle. You can create a constructor function called person and you can refer to their attributes using the 'this.' keyword.

// constructor function
function Person () {
this.name = 'John',
this.age = 23,
this.greet = function () {
console.log('hello');
}
}


Pop, Push, Shift, and Unshift

  • Pop(): remove the last item from an array

  • Shift(): removes the first item in the array

  • Unhift(): appends a new item to the beginning of the array

  • Push(): appends a new value to the end of the array




Functions

A function is a chunk of code that can be referenced by a name, and is almost like a small, self-contained mini program.

Parameters and Arguments

Parameters are set when the function is defined and Arguments invoked when the function is called.

Callbacks

A call back is simply a function that is invoked after something else happens, it deosnt just happen on its own. This happens by passing a function into another function.

  • Symbol()

    They can be used to create unique values, which helps to avoid any naming collisions.
